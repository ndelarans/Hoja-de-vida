Personal CV Landing Page

Description

This project is a personal CV landing page designed to showcase professional and academic achievements. The page includes several sections that highlight personal information, professional profile, education, skills, experience, and reading interests. This project was developed using HTML and CSS, following the requirements and criteria provided in class.

Project Structure

The project is organized into the following files and folders:

- index.html: The main HTML file for the landing page.
- styles.css: The CSS file containing the styles for the page.
- img/: A folder containing images used in the landing page.
- README.md: This file providing a brief description of the project.

Sections

1. Header
   - Includes a background image.
   - Displays personal information: photo, name, email, current semester, and a quote from a recognized figure in the technological environment.

2. About
   - Contains a paragraph describing the professional profile, strengths, and self-assessment.

3. Education
   - Lists current career and any additional courses taken or pending.

4. Skills
   - Highlights technologies known, with two favorite technologies emphasized in color and bold.

5. Experience
   - Details three projects executed during studies, including project names, descriptions, and the semester completed.

6. Reading Likes
   - Lists three favorite books with links to their summaries or consult pages.

Acceptance Criteria

- No use of flexbox, grids, or floats for positioning elements. Utilize creative selectors and properties as covered in class.
- Use the background property for the header section background image.
- Ensure the page is visually appealing with a good combination of colors and fonts.
- All text must be in English.
- Maintain well-formatted HTML and CSS code.
- Repository name format: WEB-FE-HTML-CSS-HW1-FULLNAME
- Provide the Git repository URL for submission and grant access permissions.
